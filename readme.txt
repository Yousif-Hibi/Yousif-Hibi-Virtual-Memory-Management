Virtual Memory Management 
authored by Yousif Hibi
208082081
--description
this program will implements a simulation of CPU approaches to memory. We use the paging mechanism that allows you to run a program when only part of it
Is in memory. Content memory (also called virtual memory) is divided into pages that are brought into the main memory as needed.
--------------------------------------------------------------------------------------------------------
--functions
1.sim_mem :- it a constructor for the program that will get the information like the files and size of pages
2.FIFO:- put things in the swap file by the first in first out algorithm 
3.butInMemory:- store the page from the file from the main
4.searchMemory :- search the memory for empty place 
5.load:- access a specific address for reading in main memory  
6.store:- access a specific address for writing in main memory
7.print_memory:- print the contents of te main memory 
8.print_swap:-The function prints the contents of the swap file
9.readExecutable:-read page from executable file
10.print_page_table:-Printing the table_page
11.~sim_mem:- destructor 
12.butInSwap:- search the swap and put the page from main memory to swap 
--------------------------------------------------------------------------------------------------------
